Out of the Darkness

This is a game developed for the 2025 TSA Video game ​design competition. In this game you're locked inside, try to fend off ghosts while trying to get out a mansion. It is a 3D first person online web game. Our 3d renderer is Panda3d, for more information on that go to the 'Render' page. For copyright information go to the copyright tab.
